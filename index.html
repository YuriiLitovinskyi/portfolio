RCOM Sur-Gard Translator

Application converts Dunay protocol into standard ContactID Sur-Gard protocol and sends data to Server via TCP Socket. This allows to connect Dunay devices to a lot of Centralized Monitoring Stations (Dunay protocol is supported only by Monitoring Stations of Venbest organization). Translator also allows Monitoring Station engineers to send different commands and also to control connection with each Dunay device.

App description: Electron, MongoDB, HTML, CSS, Vanilla JS

Control Business App
